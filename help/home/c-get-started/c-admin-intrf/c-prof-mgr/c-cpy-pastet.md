# Copiar y pegar un archivo como archivo adjunto{#copy-and-paste-a-file-as-an-attachment}

Pasos para copiar y pegar un archivo como archivo adjunto de correo electrónico en Microsoft Outlook.

* Haga clic con el botón secundario en la marca de verificación del archivo y haga clic en **[!UICONTROL Copy]**. En el cuerpo del mensaje de correo electrónico, pulse Ctrl+v para adjuntar el archivo.
